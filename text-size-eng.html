How to Change Text Size

With responsive web design, you can easily change the font size to suit your needs.

Keyboard Shortcuts

When browsing the Hong Kong Food Association website on a desktop computer, modern browsers allow users to press Ctrl (Command key on Mac keyboards) and +/- keys to zoom in or out, or press Ctrl and 0 to reset to default size.

On smartphones and tablets, simply swipe up/down on the screen to zoom in or out.

Menu Options

You can also change the font size through menu options. Here are the different operations for various web browsers:

  • In Chrome, select "Zoom" from the menu and press "+" to zoom in.
  • In Firefox, select "Zoom" from the "View" column and select "Zoom In". You can also zoom in/out by pressing "+" or "-" on the page menu.
  • In Microsoft Edge, select "Text size" from the "View" column and choose "Larger" or "Largest".
  • In IE, select "Zoom" from the settings icon and choose a percentage or "Larger".
  • In Opera, select "Zoom" from the "View" column and choose a percentage.
  • In Safari, select "Zoom In" from the "View" column.
